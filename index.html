Data Science Projects

  • RAG Question-Answering System

    SThe RAG PDF Question Answering System is designed to extract answers from PDF documents using Retrieval Augmented Generation. This project uses Apache Cassandra, DataStax's Astra DB as a vector database, LangChain, Python, and GPT-3.5 to deliver accurate and efficient responses to user queries.

  • Ankinator

    This project enhances exam preparation by utilizing generative AI to automate flashcard creation in the Anki program. While Anki is known for its effectiveness in promoting long-term memorization through spaced repetition, the manual process of creating flashcards can be tedious for students. Our solution automates three steps: extracting relevant content from lecture slides, formulating questions based on that content, and summarizing information for the flashcards. The method employs text-to-text models and large language models (LLMs), and its effectiveness is evaluated against ChatGPT, aiming to improve study efficiency and effectiveness for students.

  • Multi-lingual Document Retrieval

    This project focuses on multilingual document retrieval, enabling users to query documents in any language. The system is designed to effectively process diverse language inputs and return the best matching documents. By leveraging advanced natural language processing techniques, it aims to enhance accessibility and usability for users, regardless of their language proficiency, ensuring relevant information is easily retrievable across language barriers..
